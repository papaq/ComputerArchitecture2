LAB2_CA

This project describes communication among built-in
development server (it runs on localhost port 8081)
and workers (http://localhost:8081/worker)

Workers execute searching amount of a string entries
in another one, using web framework "bottle"
and ajax-requests.

Strings setting and results can be found on client
page (http://localhost:8081/client)

Developed in PyCharm Professional edition v.4.5. Python 2.7
